Delete Project


Delete project when the project is not needed any more.


  1. Select the project to remove in the Navigator.

  2. Select the Delete action in the pop-up menu or use the DEL key.

    The user will be prompted to confirm the delete.
    The user must decide if the contents of the project should be physically or logically removed.

    The physical delete will dereference all resources within the project and remove the project from the file system.
    The logical delete will dereference the project from the active workspace and all resources within the project will remain. The project can be imported back into the workspace with the action Import Project


  3. Press the Yes button

    The project is removed from the Navigator.
