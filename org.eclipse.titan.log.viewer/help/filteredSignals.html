Filtered Signals


A list of all signals that will be filtered out when displaying the MSC view.

The wildcard '*' can be used in the signal filters. It can be placed in the beginning, end or both to filter signals that ends with, starts with or contains a certain string. When no wildcards are used the signals that exactly matches (equals) the given string is filtered.


Notes regarding filtering:

The signal filter "*" will filter all signals that equals the name "*".




Button

Description

Add...

Opens a input dialog where the user enters the name of a signal that should be added to the list.

Remove

Removes the selected signal from the list.

Restore Defaults

Restores the settings to default. Default settings is an empty list.
