Import Project


The Import Project action is used for adding an existing project, that has been removed from a workspace without deleting the contents, back into the workspace.


  1. Select Import... in the File menu.

or

  1. Select the action Import... in the pop-up menu.

    The import wizard is opened.

  2. Select General.

  3. Select Existing Projects into Workspace.

  4. Press the Next button.

  5. Enter the root directory or use the standard File Browse dialog to browse for the root directory.

    All existing projects in the selected directory is presented in a list.

  6. Select one or several projects in the list to import.

  7. Press the Finish button.

    The selected project/projects is added to the Navigator.
