Color Keywords


The user can configure the color of a specified string in a message displayed in Value View as text mode. The specified strings are (case sensitive). The coloring of keywords will affect the performance.



The Value view will look like the following when coloring of Keywords is used.


Button

Description

Add...

Opens a input dialog where the user enters the string that should be added to the list.

Remove

Removes the selected string from the list.

Color

Opens a window where the user can select the color of the keyword.

Restore Defaults

Restores the settings to default.
Default settings is

use color keywords

unchecked

keywords

an empty list
