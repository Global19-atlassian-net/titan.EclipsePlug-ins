Filtered Components


A list of all components that will be filtered out when displaying the MSC view. The Main Test Component and the System Under Test can not be filtered.

The wildcard '*' can be used in the component filters. It can be placed in the beginning, end or both to filter components that ends with, starts with or contains a certain string. When no wildcards are used the components that exactly matches (equals) the given string is filtered.


Notes regarding filtering:

The component filter "*" will filter all components that equals the name "*".




Button

Description

Add...

Opens a input dialog where the user enters the name of the component that should be added to the list.

Remove

Removes the selected component from the list.

Restore Defaults.

Restores the settings to default. Default setting is an empty list.
