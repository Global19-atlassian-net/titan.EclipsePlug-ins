BASIC compilation of the plug-in
===================

 This plug-in does not need to be compiled separately.

IMPORTANT INFORMATION FOR RELEASING THE PLUG-IN:
===================================
 
To be able to use this project, you must copy the files from TTCNc3/help to their respective locations.

/*.html to the .
/docs/* to /docs
/images/* to /images/*
/info/* to /info/*

Generated .pdf documentation files like userguide.pdf must be copied to /docs

These files were removed to decrease redundancy.
Please note that in /docs there are generated files, which must be updated before every release.
